Pei 的意見不是跟 MKT 評分並列的獨立資料——她是在看過頁面已經算好的共識/分歧結果之後，才逐手法寫下的專業回應。順序不能反過來讀。

分歧度最低的 5 個手法

sd≤25，頁面已直接判定採用或避免。

對話框／聊天氣泡疊加

avg 60.0 · sd 24.8 · n=8
採用
MKT

文字包進氣泡模擬真人對話，建立臨場感、幫圖示與資訊分層 Thomaz·c04；但同手法用在調性混亂的卡一樣救不了 Thomaz·c10。

Pei

「單獨使用對話氣泡，比起使用一整頁完整的UI截圖，更容易營造出輕鬆感，Highlight 其中某個氣泡的手法，也是在模擬動態感，讓版面變得生動。」

設計手法氣泡單獨呈現，不與整頁 UI 截圖混搭同卡；用 highlight 單一氣泡模擬動態感，不要整組並列。

真人生活情境多圖拼貼

avg 51.6 · sd 25.0 · n=11

MKT

真實場景照片＋對話框疊加好看、適合堆疊情感訴求 Sylvia·c06；但同卡也被說排版太複雜、文字太多沒重點 Thomaz·c06。

Pei

「圖片跨頁呈現，會吸引人持續往右滑動而想看更多。排版的變化性創造活潑感，是很好的說故事手法。」

尚未完全對齊Pei 肯定「排版變化＝活潑」，但沒回應「太複雜」的疑慮——活潑跟雜亂的分界線仍待跟 Pei 確認（見文末待補）。

設計手法不用大面積深色 UI 背景；戲劇張力改用照片色溫營造，這就是「用對品牌」與「顯得冷」的分界。

意見不一的 8 個手法

Pei

「這是IG的grid view而非同一則輪播，所以看來混雜插畫和真人風格。細看它的版型配置，幾何分割，連同配色都有整體感，也很善用留白給予喘息空間，避免資訊過載。」

設計手法只用在「IG grid 陳列」情境，不套用在單則輪播卡；套用時三要素缺一不可——幾何分割版型、跨圖統一色板、留白呼吸感。分歧多半源自把 grid view 當輪播卡評分的載體誤判，不是手法本身差。

設計手法固定圖文配置模板，不逐卡自由變化排版；視覺焦點鎖定標題文字，不是 UI 截圖細節；色板明確排除暗紅，改用中性或暖色底。

「這種"讓照片說話"的手法，極度仰賴照片品質，不呈現任何標題與文字，更像是編輯精選的集錦。適合用來打造品牌形象，而非傳達資訊。」

設計手法只派給「品牌形象」類貼文，執行時完全不加標題／文案覆蓋層，純靠照片敘事。這條解開了「為何有高分卡沒人解釋」——用對品牌形象用途就默默拿高分，誤用在傳達資訊就變成負評。若內容目的是傳達資訊，改用其他手法。

Layer 3

收斂設計語言

Pei 的專業判斷疊在 MKT 的期望上，得出的規則。

設計方向

Warm minimalism, informative but restrained.

溫暖克制的極簡風格，資訊型內容優先於純氛圍塑造；避免冷科技感、避免暗紅色、避免大面積暗色 UI、淘汰過時玻璃質感。

色彩

  • 暖色調優先；戲劇張力靠照片色溫取代深色 UI 背景
  • 淘汰大面積暗紅（呼應品牌脫離 TrendMicro 的期望）
  • 3D 玻璃質感＋暗色調整組退場

手法分層

  • 固定採用：Chat bubble 單獨呈現、Data chart 穿插輪播、Dark mode 改用色溫
  • 明確避免：Annotated text card
  • 條件採用：Documentary 攝影限品牌形象用途；Mascot 先評估 ROI

執行細節（手法沒問題，做工決定成敗）

  • Interface light mode：UI 尺度要混搭（局部＋全貌）
  • Phone mockup：聚焦標題、避開暗紅
  • Collage/grid：抽出「幾何分割＋統一配色＋留白」單獨執行，不與風格混搭素材綁在一起評分
  • 資訊型卡片圖文比不可逼近 1:1

內容配比與人格座標

  • 50% UI＋懶人包（字少）／30% tips·data·警示（輕鬆）／20% 生活場景 UGC（真人 relatable）
  • Warm ＋ Approachable ＋ Trustworthy 為固定三角，其餘為個案風味

待補（不可猜，需另一輪跟 Pei 確認）

  • 真人生活情境拼貼：Pei 認同「排版變化＝活潑」，但 MKT 提過「太複雜＝沒重點」——變化幅度的上限沒定案
  • 原文／改寫對照卡：Pei 唯一沒回應的手法（MKT 已判定避免，非必要但補齊較完整）
  • Makoto、小美兩位 MKT 的 reflect 開放意見留空，只有 rating＋人格標籤，樣本不完整
